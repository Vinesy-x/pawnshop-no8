# 清末·故事5：功名

> 风格标签：`#讽刺` `#微恐` `#执念`
> 时长预估：20-25分钟
> 关卡数量：4关

---

## 核心设定

**表面故事**：一个老秀才来当铺，想当掉自己的"一肚子学问"，换一个功名。

**真相**：他早就死了。死了三十年，还在考。

**社会隐喻**：科举制度把人变成考试机器。有人考到白头，考到疯，考到死——死了都放不下那个"功名"。而这套制度，最终在1905年被废除，无数人一辈子的努力化为乌有。

### 关卡4：「醒」

**目标**：合成【供奉链·长明灯】+【信物链·废除科举诏书】
**难度**：困难

## 故事深层主题

### 1. 科举是一种"系统性消耗"

这个故事不是在批判"读书"，而是批判"把读书变成唯一出路"的系统：

```
『 科举本意是选拔人才。
  但当所有人都挤这条路时，
  它就变成了绞肉机。
  
  有人考上了，成了人上人。
  更多人考不上，蹉跎一生。
  
  而且......
  考上的人，学的是八股文，
  在真实世界里，可能毫无用处。
  
  这套系统，
  既浪费人才，也制造悲剧。 』
```

### 2. 执念比死亡更可怕

孙秀才死了，但他的执念没死：

```
『 他不知道自己死了。
  他不知道科举废了。
  
  他在一条不存在的路上，
  走了三十年。
  
  执念，
  是比死亡更深的囚笼。 』
```

### 3. "功名"到底是什么

孙秀才想要的"功名"，本质上是：

```
『 "光宗耀祖"——别人的认可
  "出人头地"——阶层跃升
  "让家人过好日子"——世俗成功
  
  但为了这些，
  他失去了父母、妻子、儿子。
  
  功名没得到，
  真正重要的东西，全丢了。
  
  值吗？ 』
```

### 4. 制度终结后的迷茫

1905年科举废除，无数读书人陷入迷茫：

```
『 "我除了八股文，什么都不会。"
  "我这四十年算什么？"
  
  这种迷茫，
  和今天很多人的迷茫一样——
  
  当你一直以为的"正确道路"突然消失，
  你该怎么办？ 』
```

## 设计笔记

### 微恐元素

1. **孙秀才的脚没有落地** → 一开始就暗示他是鬼
2. **街上站满了读书人的幽魂** → 不只是一个人的悲剧
3. **废弃贡院里还有鬼在考试** → 执念的可怕
4. **孙秀才清醒后眼睛是空的** → 疯狂的执念

### 反转层次

1. 老秀才来当铺 → 他是鬼
2. 他要考举人 → 科举三十年前就废了
3. 他不知道自己死了 → 他在不存在的路上走了三十年
4. 不只是他一个人 → 整条街都是被科举困住的幽魂

### 与前四个故事的对比

| 故事 | 核心意象 | 困住人的是什么 | 解脱方式 |
|------|----------|----------------|----------|
| 童养媳 | 眼泪 | 陋习 | 打破轮回 |
| 纸扎匠 | 纸人 | 遗憾 | 说出那句话 |
| 烟鬼 | 烟枪 | 毒品/产业链 | 揭露真相 |
| 金莲 | 绣花鞋 | 缠足/物化 | 还她自由 |
| **功名** | **书卷** | **科举/功名心** | **放下执念** |
