# 清末·故事2：纸扎匠的活儿

> 风格标签：`#恐怖` `#悬疑` `#交易`
> 时长预估：15-20分钟
> 关卡数量：3关

---

## 核心交易

**客人想要**：让死去的儿子"复活"
**客人愿意付出**：四十年手艺 / 剩余阳寿 / 自己的命

**玩家选择收取什么，决定不同结局**

---

## 第一幕：入店

### 场景：黄昏的当铺

**【环境渲染】**

```
『 黄昏时分。
  
  夕阳的余晖照进当铺，
  把柜台染成了橘红色。
  
  像是......血的颜色。
  
  风铃响了。 』
```

**【客人登场】**

*一个干瘦的老头走了进来。*
*他的手指很长，关节突出，指甲缝里塞满了浆糊和纸屑。*

```
『 他弓着背，
  像一只瘦骨嶙峋的螳螂。
  
  眼睛很亮，
  亮得有些瘆人。 』
```

**【初次对话】**

**老陈**（纸扎匠）
> "掌柜的，我来做个买卖。"
> 
> *他从背后的竹篓里取出一个布包。*
> 
> *打开——是一双手。纸扎的手，精细得像真人一样。*

```
（ 这双手......
   纹路、指甲、甚至掌纹，
   都和真手一模一样。 ）
```

**老陈**
> "这是我四十年的手艺。"
> 
> "我要用它换一样东西——"
> 
> *他的眼神变得炽热。*
> 
> "我要换我儿子的命。"

```
（ 以艺换命......
   这笔买卖，可不简单。 ）
```

**掌柜开口**（旁白选项，玩家选择）
```
┌─────────────────────────────────────────┐
│  你想换什么？                            │
│                                         │
│  [手艺换不了命]                          │
│  [说说你儿子的事]                        │
└─────────────────────────────────────────┘
```

*选择后继续：*

**老陈**
> "手艺换不了？那我还有别的——"
> 
> "我的阳寿，我的命，都可以给你！"
> 
> *他跪了下来。*
> 
> "求掌柜的成全！"

```
『 他把头磕在地上，
  "砰砰"作响。
  
  这份执念......
  重得像山。 』
```

**【进入关卡提示】**

```
『 先看看这笔买卖的分量。
  
  他的儿子，究竟是怎么死的？ 』
```

**[进入关卡1]**

## 第二幕：探寻

### 关卡1：「纸上功夫」

**目标**：合成【信物链·染血家书】
**难度**：简单
**特殊机制**：无

---

### 剧情碎片A：白发人送黑发人

*关卡结束，画面渐暗。*

```
『 老陈的儿子叫陈大顺。
  
  去年冬天出门做生意，
  遇上了土匪。
  
  劫财还要害命。
  
  等老陈找到儿子时，
  尸体已经冻硬了。 』
```

**画面**：*雪地上，一个年轻人倒在血泊中。*

```
『 老陈没哭。
  
  他把儿子背回家，
  亲手给他做了一套纸扎——
  纸房子、纸马车、金童玉女。
  
  那是他做过最好的活儿。 』
```

**【回到当铺】**

**老陈**
> "我给别人扎了一辈子，"
> "轮到我儿子，我也没舍得省工。"
> 
> *他的声音发涩。*
> 
> "可是掌柜的，烧完之后......"
> "我做了个梦。"

```
（ 梦？ ）
```

**老陈**
> "我梦见大顺了。"
> 
> "他说——'爹，我收到了，但我回不去。'"
> 
> "他说——'除非有人愿意换。'"
> 
> *老陈的眼睛亮了起来。*
> 
> "掌柜的，我愿意换！"

**[进入关卡2]**

---

### 关卡2：「痴人说梦」

**目标**：合成【供奉链·纸扎童子】
**难度**：中等
**特殊机制**：棋盘上有「纸屑」障碍

---

### 剧情碎片B：地窖里的秘密

*关卡结束，画面切换到老陈家——*

```
『 你的意识跟着他回到了家。
  
  老旧的院子，堆满了纸和竹篾。
  
  还有......一个地窖。 』
```

**画面**：*幽暗的地窖，火把照亮墙壁。*

```
『 墙边靠着一排纸人。
  
  真人大小，穿着陈大顺生前的衣服。
  
  一个、两个、三个......
  
  七个"陈大顺"。 』
```

**画面**：*七张一模一样的脸，但眼睛都是空洞的。*

```
『 老陈每天都在扎。
  
  扎一个，不像，烧掉。
  再扎一个，还是不像，再烧。
  
  这七个，是他舍不得烧的。
  
  因为越来越像了。 』
```

**【回到当铺】**

*你看向老陈。*

```
（ 所以......
   他不只是想换儿子的命。
   他想把儿子「扎」回来。 ）
```

**老陈**
> "掌柜的，你都看到了。"
> 
> *他举起那双纸手。*
> 
> "这是第八次了。手已经扎好了。"
> 
> "只差一口气。"
> 
> "我用我的命换——让这双手能动起来，行不行？"

**[进入关卡3]**

---

## 第三幕：真相

### 关卡3：「还魂之术」

**目标**：合成【药理链·还魂香】
**难度**：较高
**特殊机制**：棋盘边缘有「纸人手」伸出干扰

---

### 剧情碎片C：儿子的真相

*关卡结束。*
*还魂香成型的瞬间，你看到了另一段记忆——*

```
『 那是陈大顺出门前的夜晚。 』
```

**画面**：*年轻人在收拾行囊，老陈在旁边絮叨。*

```
『 "路上小心，别走小道。"
  "钱财是小，命是大。"
  
  陈大顺笑着应了。
  
  但他的眼神......
  有些奇怪。 』
```

**画面**：*大顺的手在发抖。*

```
『 他知道。
  
  他早就知道那条路不安全。
  
  但他还是去了。
  
  因为...... 』
```

**画面**：*一封信。上面写着：老陈欠债，三日内还清，否则拿命来抵。*

```
『 老陈年轻时赌过钱。
  
  欠下的债，利滚利，滚成了天文数字。
  
  债主找上门，给了三天期限。
  
  陈大顺没告诉爹。
  
  他只说：我出门做趟买卖，很快回来。 』
```

*画面定格：大顺回头看了老陈一眼，像是在告别。*

```
『 他不是被土匪杀的。
  
  是被债主杀的。
  
  是替他爹......死的。 』
```

**【回到当铺】**

*老陈的脸色惨白。*

**老陈**
> "不......不是的......"
> 
> "那些债，我早就还清了......"
> 
> "大顺他......他不知道的......"
> 
> *他浑身发抖。*
> 
> "他是因为我死的？"
> "是我害死了他？！"

```
『 真相比死亡更沉重。
  
  他想复活儿子，
  却发现儿子是替自己死的。
  
  这笔债......
  怎么还？ 』
```

## 第四幕：抉择

**【掌柜内心独白】**

```
（ 他说要换儿子的命。
   
   可现在他知道了——
   儿子的命，本来就是替他还的债。
   
   这笔交易......
   
   我该收他什么？ ）
```

**【抉择呈现】**

```
这笔买卖，你要收什么？

老陈跪在地上，等待你的回答。
```

---

### 选项A：收取「四十年手艺」

> **「你的手艺，从今往后，再也扎不出像样的东西。」**
> 
> *以艺换子。*

---

### 选项B：收取「剩余阳寿」

> **「你还剩多少阳寿，就拿多少来换。」**
> 
> *以命换命。*

---

### 选项C：收取「那笔债的记忆」 🔒

> **「你忘掉那笔债，也忘掉儿子是怎么死的。」**
> 
> *需要道具：天机符*

---

## 第五幕：结局

### 结局A：「废手」

*老陈点了点头。*

**老陈**
> "好......好。"
> 
> "手艺没了就没了。"
> 
> "只要能见大顺一面......"

```
『 你将还魂香点燃。
  
  老陈手中的纸手，
  微微动了一下。 』
```

*老陈狂喜地抓住那双手。*

**老陈**
> "大顺！大顺！"

```
『 纸手在他掌心轻轻握了握。
  
  然后......
  缓缓散开，化成了纸屑。
  
  那是陈大顺最后的告别。 』
```

*老陈愣在原地。*

**老陈**
> "就......就这样？"
> 
> *他的声音在发抖。*
> 
> "我只能......握一下他的手？"

```
（ 一条命，
   换一次握手。
   
   公平吗？
   
   也许......这就是告别的意义。 ）
```

```
『 从此以后，
  老陈的手再也扎不出像样的东西。
  
  但他没有怨言。
  
  他把那些纸屑收好，
  做成了一个小小的纸人，
  放在儿子的牌位旁边。
  
  那是他最后的作品。
  
  虽然粗糙，但像极了大顺小时候的样子。 』
```

**【收集物品】**
- 「一双纸手（化作纸屑）」入库

**【收集室文案】**
> 这些纸屑曾是一双手。
> 
> 纸扎匠老陈用四十年的手艺，换来了和儿子最后的握手。
> 
> 只有一瞬间。
> 
> 但他说，够了。

**获得修缮点：⭐⭐⭐**

---

### 结局B：「偿命」

*老陈沉默了一会，然后慢慢点头。*

**老陈**
> "行。"
> 
> "他替我死的，我替他活回来。"
> 
> "公平。"

```
『 你将还魂香点燃。
  
  老陈的身体开始发光，
  阳寿一点一点地流逝。
  
  而那双纸手......
  开始有了温度。 』
```

*纸手慢慢变成血肉之手。*
*然后是手臂、身体、脸。*

*陈大顺站在当铺里，和活人一模一样。*

**陈大顺**
> "爹......？"
> 
> *他看向老陈——*
> 
> *老陈已经变成了一具干瘪的尸体。*

**陈大顺**
> "爹！！"

```
『 老陈用尽阳寿，
  换回了儿子的命。
  
  临死前他笑了。
  
  "大顺啊，你替爹死过一回了。"
  "这回，换爹替你死。"
  "扯平了。" 』
```

*陈大顺跪在父亲尸体旁，放声大哭。*

```
『 半年后。
  
  陈大顺接过了父亲的手艺。
  
  他扎的纸活儿，
  比父亲还要好。
  
  也许是因为，
  他真的去过那边。 』
```

**【收集物品】**
- 「一双纸手（已化为血肉）」入库
- 「老陈的剪刀」入库（大顺继承的遗物）

**【收集室文案】**
> 这双手曾是纸扎的。
> 
> 老陈用全部阳寿，把它变成了血肉。
> 
> 儿子活了。父亲死了。
>
> 债，清了。

**获得修缮点：⭐⭐⭐（+⭐隐藏奖励）**

---

### 结局C：「忘债」

*需要道具：天机符*

**掌柜**（旁白）
```
（ 也许最沉重的，
   不是死亡，
   是知道真相后的愧疚。
   
   如果忘掉那笔债......
   他能好受些吗？ ）
```

*你取出天机符。*

**掌柜**
> "我收你那笔债的记忆。"
> 
> "你会忘掉欠债的事，也忘掉你儿子是怎么死的。"
> 
> "你只会记得——他被土匪杀了。"
> 
> "这样，你换不换？"

*老陈愣住了。*

**老陈**
> "忘掉......？"
> 
> "忘掉是我害死他的？"
> 
> *他的眼泪流了下来。*
> 
> "可是我不想忘。"
> 
> "这是我欠他的。"
> 
> "我忘了，谁来记得他是个好儿子？"

```
『 他没有接受交易。
  
  他选择带着愧疚活下去。 』
```

**老陈**
> "掌柜的，这笔买卖......我不做了。"
> 
> *他捧起那双纸手。*
> 
> "大顺替我死，我不能再让他替我活。"
> 
> "我就带着这份债，活到自己该死的那天。"
> 
> "然后......亲自去找他还。"

```
『 他站起身，
  佝偻的背似乎挺直了一些。
  
  "多谢掌柜让我知道真相。"
  "这笔人情，我记下了。"
  
  他离开了当铺。
  
  地窖里的七个纸人，
  当晚就被他一把火烧了。 』
```

*三年后。*

```
『 老陈死了。
  
  死前他把所有家产捐给了镇上的孤儿。
  
  他说：
  "大顺心善，他肯定希望我这么做。"
  
  入殓的时候，
  那双纸手放在他胸口。
  
  手指的姿势，像是在握着什么。
  
  也许是另一只手吧。 』
```

**【收集物品】**
- 「一双纸手」入库
- 「老陈的忏悔书」入库（写给儿子的遗书）
- 「孤儿院匾额」入库（隐藏——上书"大顺堂"）

**【收集室文案】**
> 这双手是老陈最后的作品。
> 
> 他没有用它换回儿子，而是带着它，带着愧疚，活到了最后。
> 
> 死前他建了一座孤儿院，取名"大顺堂"。
>
> 债，不是用命还的。是用活着的每一天还的。

**获得修缮点：⭐⭐⭐⭐⭐（完美结局）**

## 设计笔记

### 当铺作用
- **典当物**：手艺 / 阳寿 / 记忆
- **换取物**：儿子复活的不同程度
- **玩家抉择**：收什么决定结局

### 反转设计
- 以为是土匪杀人
- 真相是儿子替父还债而死
- 愧疚比死亡更重

### 三个结局的差异
- A：小团圆（握手告别）
- B：命换命（父死子活）
- C：赎罪（带着愧疚活下去）
