# 清末·故事2：纸扎匠的活儿

> 风格标签：`#恐怖` `#父爱` `#交易`
> 时长预估：15-20分钟
> 关卡数量：3关

---

## 核心设定

老陈是纸扎匠，儿子大顺死于瘟疫。

老陈听说当铺能做「交易」，想用自己的东西换儿子回来。

**他不懂什么法术，他只会扎纸——所以他扎了一具完美的纸躯，想让儿子的魂有个「身体」可以附。**

---

## 核心交易

**客人想要**：让儿子的魂魄回来，附在纸躯上
**客人愿意付出**：四十年手艺 / 自己的阳寿 / 自己替儿子去死

**玩家选择收取什么，决定不同结局**

```
『 他弓着背，
  眼睛却亮得瘆人。 』
```

> "掌柜的，我来做笔买卖。"

**老陈**

> "我想......我能不能把他扎回来？"

> "哪怕只有一天，让我再和他说说话......"

> 
> "掌柜的，我什么都可以给！"

## 第三幕：真相

### 关卡3：「招魂」

**目标**：合成【药理链·还魂香】
**难度**：较高
**特殊机制**：棋盘边缘有「纸人手」伸出干扰

---

### 剧情碎片C：大顺的遗愿

*关卡结束。*
*还魂香成型的瞬间，你看到了一段画面——*

```
『 那是大顺临死前的夜晚。 』
```

**画面**：*病榻上，年轻人面色苍白，老陈守在床边。*

**大顺**（虚弱地）
> "爹......我怕是不行了。"

**老陈**
> "别瞎说！明天大夫还来呢！"

**大顺**
> "爹，我有一件事......一直没跟你说。"
> 
> *他艰难地抬起手。*
> 
> "我......其实挺想学扎纸的。"

```
『 老陈愣住了。 』
```

**大顺**
> "小时候看您扎纸人，觉得真神。"
> 
> "后来......后来我装作不感兴趣，是怕被人笑话。"
> 
> *他咳嗽起来，咳出了血。*
> 
> "爹......如果我还能活，我一定跟您学......"

```
『 那是大顺说的最后一句话。
  
  当晚，他就走了。
  
  老陈甚至没来得及说：
  
  "好，爹教你。" 』
```

**【回到当铺】**

*老陈的眼泪流了下来。*

**老陈**
> "他想学......他一直想学......"
> 
> "我还骂他没出息......"
> 
> *他抱着纸人，像抱着自己的儿子。*
> 
> "大顺，爹现在就教你，好不好？"
> 
> "只要你回来......爹什么都教你......"

```
『 一个父亲最大的遗憾——
  
  是没能在儿子活着的时候，
  告诉他：
  
  "好，爹教你。" 』
```

## 第五幕：结局

### 结局A：「废手」

*老陈毫不犹豫。*

**老陈**
> "好。"
> 
> "手艺没了就没了。"
> 
> "我这辈子扎的够多了。"

```
『 你将还魂香点燃，
  青烟缭绕，飘向纸人。
  
  纸人的眼睛......动了。 』
```

*老陈狂喜。*

**老陈**
> "大顺！大顺！"

**纸人大顺**
> "......爹？"
> 
> *声音沙哑，像纸张摩擦。*
> 
> "我......我在哪？"

```
『 他回来了。
  
  魂附纸躯，
  不是活人，但能说话，能动。
  
  老陈教他扎纸，
  从最简单的灯笼开始。
  
  纸做的手指笨拙地折着纸——
  
  "爹，我是不是很笨？"
  
  "不笨，我儿子最聪明。" 』
```

*七天后。*

```
『 还魂香的效力到了尽头。
  
  纸人大顺慢慢僵硬，
  不再动了。
  
  但他的脸上带着笑。
  
  手里还攥着一个纸灯笼——
  他学会的第一个作品。 』
```

**老陈**
> "大顺，你做得真好。"
> 
> *他把灯笼收好。*
> 
> "比爹当年还好。"

```
『 从此以后，
  老陈的手再也扎不出像样的东西。
  
  但他不后悔。
  
  那个纸灯笼，
  他挂在堂屋里，
  每晚都点亮。 』
```

**【收集物品】**
- 「纸人大顺」入库
- 「一盏纸灯笼」入库

**【收集室文案】**
> 这盏灯笼是一个纸人做的。
> 
> 也是他做的唯一一件东西。
> 
> 纸扎匠老陈说：这是他儿子的作品，比他当年做得都好。

**获得修缮点：⭐⭐⭐**

---

### 结局B：「折寿」

*老陈点头。*

**老陈**
> "十年就十年。"
> 
> "我老了，活那么久做什么。"

```
『 你将还魂香点燃。
  
  老陈的头发瞬间白了一半，
  脸上的皱纹更深了。
  
  而纸人......睁开了眼。 』
```

*纸人大顺站了起来。*

**纸人大顺**
> "爹......？"
> 
> *他低头看看自己的手——纸做的。*
> 
> "我......我怎么变成这样了？"

**老陈**
> "大顺，别怕。"
> 
> *他握住纸做的手。*
> 
> "爹在呢。"

```
『 十年阳寿换来的，
  是更长久的陪伴。
  
  纸人大顺不会老，不会病，
  只是不能出门见人。
  
  老陈每天和他在一起，
  教他扎纸，陪他说话。
  
  邻居们说老陈疯了，
  整天对着一个纸人自言自语。
  
  老陈只是笑笑。
  
  "你们不懂。" 』
```

*十年后。*

```
『 老陈死了。
  
  死的时候，
  纸人大顺跪在床边，
  一动不动。
  
  眼眶里......流出了纸屑。
  
  像是眼泪。 』
```

```
『 入殓那天，
  邻居发现纸人不见了。
  
  只有老陈的棺材里，
  多了一个小小的纸人——
  
  是大顺扎的。
  
  一个抱着孩子的老父亲。 』
```

**【收集物品】**
- 「纸人大顺」入库
- 「纸父子像」入库（隐藏——大顺扎的）

**【收集室文案】**
> 这个小纸人是一个纸人扎的。
> 
> 一个儿子送给父亲的最后礼物。
> 
> 纸人会扎纸，这件事没人信。但老陈信。

**获得修缮点：⭐⭐⭐（+⭐隐藏奖励）**

---

### 结局C：「遗言」

*需要道具：天机符*

**掌柜**
> "我不收你的手艺，也不收你的阳寿。"
> 
> "我只要你把那句话说出来。"
> 
> "那句你一直没来得及说的话。"

*老陈愣住了。*

**老陈**
> "什么话......？"

**掌柜**
> "你儿子想学扎纸。"
> 
> "你还没说'好'。"

```
『 你将天机符贴在纸人额头。
  
  纸人的眼睛亮了一瞬——
  
  是大顺的魂，短暂地回来了。 』
```

**纸人大顺**
> "爹......？"

*老陈浑身颤抖。*

**老陈**
> "大顺......大顺！"
> 
> *他扑过去，抱住纸人。*
> 
> "爹在！爹在！"

**纸人大顺**
> "爹，我......我是不是快走了？"
> 
> *声音很轻，像风吹过纸张。*
> 
> "我好像看到路了......"

*老陈拼命摇头。*

**老陈**
> "别走！别走！"
> 
> "大顺，爹还没......"
> 
> *他哽咽了。*
> 
> "爹还没教你扎纸呢......"

**纸人大顺**
> "爹......"
> 
> *他笑了。*
> 
> "您说'好'就行了。"
> 
> "我在那边，自己学。"

```
『 老陈哭着喊出了那句话——
  
  "好！爹教你！爹什么都教你！"
  
  纸人大顺的笑容定格在脸上，
  光芒散去，
  重新变回了纸。
  
  但那笑容......
  是满足的。 』
```

*老陈跪在地上，嚎啕大哭。*

```
『 那天之后，
  老陈把纸人供在堂屋。
  
  每天，他都会在纸人面前，
  扎一件小东西，
  一边扎一边讲：
  
  "大顺，你看，这是灯笼......"
  "这是糊法，你记住了......"
  "这是你最喜欢的老虎......"
  
  像是在教一个永远不会回应的学生。
  
  又像是在跟一个永远都在的儿子说话。 』
```

*三年后。*

```
『 老陈死了。
  
  死在纸人面前，
  手里还握着一个没做完的小纸人。
  
  邻居说，
  他脸上带着笑。
  
  像是终于把想教的都教完了。 』
```

**【收集物品】**
- 「纸人大顺」入库
- 「一句迟到的话」入库（上书一个字："好"）
- 「纸老虎」入库（隐藏——老陈最后在做的那个）

**【收集室文案】**
> 这个"好"字，迟到了一辈子。
> 
> 但说出来的那一刻，一切都值了。
> 
> 老陈最后三年，每天都在"教"儿子扎纸。没人知道大顺学会了没有。
>
> 但老陈说："我儿子聪明，肯定学会了。"

**获得修缮点：⭐⭐⭐⭐⭐（完美结局）**

## 设计笔记

### 核心情感
- 父子之间的遗憾：没来得及说的话
- 死前想学扎纸 → 死后父亲想教
- 七个失败品 = 父亲对儿子模样的执念

### 当铺交易逻辑
- 手艺换短暂陪伴
- 阳寿换长久陪伴
- 一句话换一个告别

### 恐怖元素
- 地窖里的失败品
- 纸人的眼睛会动
- 纸人流出的「纸屑眼泪」
