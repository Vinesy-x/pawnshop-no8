# 清末·故事2：纸扎匠的活儿

> 风格标签：`#恐怖` `#悬疑` `#反转`
> 时长预估：15-20分钟
> 关卡数量：3关

---

## 故事背景

清末，纸扎行当兴盛。丧葬用的纸人、纸马、纸屋，都要找纸扎匠来做。

手艺好的匠人，扎出来的东西栩栩如生——

**太像了，反而让人害怕。**

---

## 第一幕：入店

### 场景：黄昏的当铺

**【环境渲染】**

```
『 黄昏时分。
  
  夕阳的余晖照进当铺，
  把柜台染成了橘红色。
  
  像是......血的颜色。
  
  风铃响了。 』
```

**【客人登场】**

*一个干瘦的老头走了进来。*
*他的手指很长，关节突出，指甲缝里塞满了浆糊和纸屑。*

```
『 他弓着背，
  像一只瘦骨嶙峋的螳螂。
  
  眼睛很亮，
  亮得有些瘆人。 』
```

**【初次对话】**

**老陈**（纸扎匠）
> "掌柜的，我来当个东西。"
> 
> *他从背后的竹篓里，小心翼翼地取出一个布包。*
> 
> *打开——是一双手。*
> 
> *纸扎的手，但精细得像真人一样。*

```
（ 这双手......
   纹路、指甲、甚至掌纹，
   都和真手一模一样。
   
   不，比真手更精致。 ）
```

**老陈**
> "这是我的手艺。"
> 
> "我做了四十年纸扎，这是我最好的作品。"
> 
> *他的眼神有些飘忽。*
> 
> "我想当掉它，换一个人。"

```
（ 换一个人？
   什么意思？ ）
```

**老陈**
> "我儿子......我要把我儿子换回来。"
> 
> *他突然激动起来。*
> 
> "他才三十岁啊！怎么能走在我前头！"
> 
> "掌柜的，我听说你这里什么都能换......"
> 
> "我用我的手艺，换他回来，行不行？"

```
『 他的眼眶红了，
  但没有眼泪。
  
  也许是哭干了。 』
```

**【进入关卡提示】**

```
『 以生换生......
  
  这笔交易，代价不小。
  
  先看看，他的手艺究竟值多少。 』
```

**[进入关卡1]**

---

## 第二幕：探寻

### 关卡1：「纸上功夫」

**目标**：合成【信物链·绣花鞋】
**难度**：简单
**特殊机制**：无

---

### 剧情碎片A：手艺人

*关卡结束，画面渐暗。*
*出现泛黄的回忆画面。*

```
『 老陈十二岁学艺，
  从糊灯笼开始，
  到扎纸人、纸马、纸房子。
  
  师父说他有天赋——
  "你扎的东西，像是有魂。" 』
```

**画面**：*年轻的老陈在扎纸人，纸人的眼睛栩栩如生。*

```
『 二十岁那年，他出师了。
  镇上的丧事都找他。
  
  "陈师傅扎的纸人，
   烧过去，准能收到。"
  
  他娶了媳妇，生了儿子，
  日子过得还算安稳。 』
```

**【回到当铺】**

**老陈**
> "我这辈子，就会扎纸活儿。"
> 
> "什么金童玉女、高头大马，我都做得来。"
> 
> *他低下头。*
> 
> "可我扎了一辈子给死人的东西，"
> "却留不住自己的儿子......"

```
（ 他的儿子是怎么死的？ ）
```

**[进入关卡2]**

---

### 关卡2：「命中注定」

**目标**：合成【供奉链·纸扎童子】
**难度**：中等
**特殊机制**：棋盘上有「纸屑」障碍，会随机移动

---

### 剧情碎片B：儿子的死

*关卡结束，画面渐暗。*
*一个年轻男人的身影出现——那是老陈的儿子。*

```
『 儿子叫陈大顺，
  人如其名，什么都顺。
  
  会读书，会做生意，
  还娶了个好媳妇。
  
  老陈常说：
  "我这辈子最得意的作品，
   不是纸扎，是我儿子。" 』
```

**画面**：*年轻人笑着搀扶老陈。*

```
『 去年冬天，大顺出门做生意。
  
  本来说好七天回来，
  却再也没回来。
  
  土匪。
  
  劫财还要害命。 』
```

**画面**：*一条泥泞的路，雪地上有暗红的痕迹。*

```
『 等老陈找到儿子的时候，
  尸体已经冻硬了。
  
  他亲手给儿子扎了一整套——
  纸房子、纸马车、金童玉女。
  
  那是他做过最好的纸扎。 』
```

**【回到当铺】**

*老陈的声音沙哑了。*

**老陈**
> "我给别人扎了一辈子，"
> "到头来......轮到我儿子了。"
> 
> *他攥紧那双纸手。*
> 
> "掌柜的，你说这叫什么事？"
> 
> "我就这么一个儿子啊......"

```
（ 白发人送黑发人。
   
   天底下最苦的事，
   莫过于此。 ）
```

**[进入关卡3]**

---

## 第三幕：真相

### 关卡3：「还魂之术」

**目标**：合成【药理链·还魂香】
**难度**：较高
**特殊机制**：棋盘边缘会有「纸人」伸出手，干扰操作

---

### 剧情碎片C：地窖里的秘密

*关卡结束。*
*还魂香成型的瞬间，画面突然一闪——*

```
『 你看到了老陈家的地窖。
  
  那里......不对劲。 』
```

**画面**：*幽暗的地窖，火把的光照亮墙壁。*

```
『 墙边靠着一排纸人。
  
  不，不是普通的纸人——
  
  那是......
  
  真人大小的纸扎人。 』
```

**画面**：*镜头拉近，纸人的脸——是一个年轻男人的脸。*

```
『 那张脸，和陈大顺一模一样。
  
  一个，两个，三个......
  
  地窖里有七个"陈大顺"。 』
```

*画面回到当铺。*
*老陈的表情变了。*

**老陈**
> "你......你看到了？"
> 
> *他的嘴角抽搐了一下。*
> 
> "那些是......失败品。"
> 
> "我想把大顺扎回来。"
> "给他扎一个身子，让他能回来。"
> 
> *他举起那双纸手。*
> 
> "这是第八次了。"
> "这一次，我终于扎好了他的手......"

```
『 你突然明白了什么。
  
  他不是来当手艺的。
  
  他是想用纸......
  把死去的儿子「扎」回来。 』
```

**老陈**
> "掌柜的，求你了——"
> 
> *他跪了下来。*
> 
> "我只差一口气了。"
> "只要有还魂香，我就能让他......"
> 
> *他的眼神有些疯狂。*
> 
> "让他动起来！"

```
（ 纸扎的人......
   真的能「活」过来吗？
   
   就算活了，
   那还是他的儿子吗？ ）
```

---

## 第四幕：抉择

**【掌柜内心独白】**

```
（ 四十年的手艺，
   只为扎一个儿子。
   
   这份执念，让人心酸。
   
   可是——
   
   纸做的人，终究不是人。
   就算披了人皮，里面也是空的。
   
   让他"成功"，是帮他，还是害他？
   
   掌柜啊掌柜，
   这笔账，你怎么算？ ）
```

**【抉择呈现】**

```
他在等待你的回答。

还魂香就在你手中。
```

---

### 选项A：完成交易

> **「这是你的执念，你自己了断。」**
> 
> *将还魂香交给他。*

---

### 选项B：拒绝交易

> **「纸人不是人。」**
> 
> *将还魂香收回。*

---

### 选项C：另辟蹊径 🔒

> **「让你儿子自己说。」**
> 
> *需要道具：天机符*

---

## 第五幕：结局

### 结局A：「纸人复生」

*老陈颤抖着接过还魂香。*

**老陈**
> "谢谢......谢谢掌柜的！"
> 
> *他连滚带爬地跑出了当铺。*

```
『 三天后。
  
  镇上传出一个诡异的消息——
  
  有人看到陈大顺了。
  
  他站在自己的坟前，
  一动不动。 』
```

**画面**：*月光下，一个年轻人的背影站在坟前。*

```
『 老陈疯了一样跑过去。
  
  "大顺！大顺！"
  
  那人慢慢转过身来。
  
  一张和陈大顺一模一样的脸——
  
  但眼睛是空洞的。 』
```

**画面**：*纸人的脸，眼窝里是两个黑洞。*

```
『 "爹......"
  
  它开口了。
  声音像风吹过竹管，空洞洞的。
  
  "爹......我好冷。"
  
  老陈愣住了。
  
  然后......他笑了。
  
  "不冷，爹给你扎件棉袄。"
  "大顺乖，爹带你回家。" 』
```

*画面渐黑。*

```
『 从此以后，
  老陈再也没出过门。
  
  邻居们有时会听到他在屋里说话，
  像是在和儿子聊天。
  
  但从来没有第二个声音回应。
  
  只有......纸张摩擦的沙沙声。 』
```

**【收集物品】**
- 「一双纸手」入库

**【收集室文案】**
> 这双手来自纸扎匠老陈。
> 
> 他用四十年的手艺，"复活"了自己的儿子。
> 
> 据说，他至今还住在那间老屋里。
> 
> 和他的"儿子"在一起。
> 
> 没人敢靠近。

**获得修缮点：⭐⭐⭐**

---

### 结局B：「破碎执念」

*老陈愣住了。*

**老陈**
> "不是人？"
> 
> "我知道......我当然知道......"
> 
> *他低下头看着那双纸手，肩膀开始颤抖。*
> 
> "可我就是想再看他一眼啊......"

**掌柜**（旁白）
```
（ "你扎的是纸，不是人。"
   "就算它能动、能说话，"
   "那也不是你的大顺。"
   "只是一个......皮囊。" ）
```

*老陈的手在发抖。*

**老陈**
> "皮囊......"
> 
> *他惨笑一声。*
> 
> "我给人扎了一辈子皮囊，"
> "到头来，想给自己儿子扎一个，"
> "也不行吗？"

*沉默了很久。*

**老陈**
> "罢了......"
> 
> *他把那双纸手轻轻放在柜台上。*
> 
> "掌柜的，这双手你收着吧。"
> 
> "我......我得回去，把那些烧了。"

```
『 他佝偻着背，走出了当铺。
  
  那天晚上，老陈家着了一场大火。
  
  地窖里的七个纸人，
  全烧成了灰。
  
  老陈站在火边，看着火光，
  一滴泪也没有。 』
```

*三个月后。*

```
『 老陈死了。
  
  死在儿子的坟前。
  
  脸上挂着笑，
  手里攥着一个拨浪鼓——
  那是大顺小时候玩过的。
  
  也许在梦里，
  他终于见到儿子了吧。 』
```

**【收集物品】**
- 「一双纸手」入库
- 「一只拨浪鼓」入库（隐藏）

**【收集室文案】**
> 这双手来自纸扎匠老陈。
> 
> 他本想用它"复活"儿子，最终选择了放手。
> 
> 他把毕生的作品付之一炬，
> 然后去了儿子的坟前，
> 再也没有起来。
>
> 死得很安详。

**获得修缮点：⭐⭐⭐（+⭐隐藏奖励）**

---

### 结局C：「亡者之言」

*需要道具：天机符*

**掌柜**（旁白）
```
（ 他想见儿子，
   也许......让他真的"见"一次，
   比什么都强。 ）
```

*你取出天机符，贴在那双纸手上。*

```
『 纸手微微发光。
  
  一个模糊的人影从中浮现——
  
  那是陈大顺的魂。 』
```

*老陈瞪大了眼睛。*

**老陈**
> "大顺！大顺！"
> 
> *他伸手想去抱，却穿过了虚影。*
> 
> "儿啊......爹来接你回家！"

*陈大顺的魂静静地看着他。*

**陈大顺**
> "爹。"
> 
> *声音很轻，像是从很远的地方传来。*
> 
> "您别扎了。"

**老陈**
> "大顺......？"

**陈大顺**
> "我在那边挺好的。"
> 
> *他笑了笑。*
> 
> "您扎的房子、马车，我都收到了。"
> 
> "够用了，爹。"
> 
> "您别再扎了。"

*老陈浑身颤抖。*

**老陈**
> "可是......可是爹想你啊......"

**陈大顺**
> "我知道。"
> 
> *他的身影开始变淡。*
> 
> "爹，您好好活着。"
> 
> "等您的时候到了，我来接您。"
> 
> "到时候......咱爷俩好好喝一杯。"

```
『 陈大顺的魂渐渐消散，
  像是从没出现过。
  
  只剩下那双纸手，
  静静地躺在柜台上。 』
```

*老陈跪在地上，嚎啕大哭。*

**老陈**
> "好......好......"
> 
> "爹等着......爹等着你来接......"

```
『 那是他儿子死后，
  第一次哭出声来。
  
  哭完之后，
  他把那双纸手收了起来。
  
  没有烧掉。
  
  他说，这是他和儿子的约定。
  等他死的那天，
  要带着这双手一起走。 』
```

*两年后。*

```
『 老陈老死在床上。
  
  走的时候，
  怀里抱着那双纸手，
  脸上带着笑。
  
  据说入殓的时候，
  那双纸手不知怎的，
  自己动了一下——
  
  像是在握他的手。 』
```

**【收集物品】**
- 「一双纸手」入库
- 「陈大顺的信」入库（魂魄留下的，上面只有一个字："等"）
- 「父子酒杯」入库（隐藏——两只纸扎的酒杯，是老陈死后发现的）

**【收集室文案】**
> 这双手来自纸扎匠老陈。
> 
> 他用它"见"了儿子最后一面。
> 
> 他们约定，来生再聚。
>
> 据说老陈下葬那天，有人在坟前看到两道身影——
> 一老一少，并肩而行，渐渐消失在夕阳里。

**获得修缮点：⭐⭐⭐⭐⭐（完美结局）**

## 设计笔记

### 恐怖元素
- 地窖里的七个「失败品」
- 纸人的空洞眼睛
- 「纸张摩擦的沙沙声」

### 温情元素
- 父子之间的羁绊
- 陈大顺的魂让父亲放手
- 约定来生再聚

### 主题
- 执念能让人疯狂
- 但有些离别，只能接受
- 爱的最好方式，是放手
