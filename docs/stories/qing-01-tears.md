# 清末·故事1：童养媳的眼泪

> 风格标签：`#悲情` `#复仇` `#抉择`
> 时长预估：15-20分钟
> 关卡数量：3关

---

## 第一幕：入店

### 场景：夜晚的当铺

**【环境渲染】**

```
『 子时三刻。

  当铺外飘着细雨，
  檐下的灯笼被风吹得摇摇晃晃。
  
  风铃响了。
  
  有客人来了。 』
```

**【客人登场】**

*画面渐亮，一个身影站在门口。*

```
『 她站在那里，
  像是站了很久。
  
  粗布衣裳，
  驼着的背，
  满是冻疮的手。
  
  明明是个年轻女子，
  脸上却刻满了皱纹。 』
```

**【初次对话】**

**阿莲**（客人名字，此时玩家不知道）
> "掌柜的......这里是当铺吧？"

**掌柜**（玩家视角，不出声，以旁白呈现）
```
（ 她的声音沙哑，
   像是很久没有开口说过话。 ）
```

**阿莲**
> "我......我有东西要当。"
> 
> *她从怀里掏出一个陶罐，小心翼翼地捧着。*
> 
> "这是我三十年的眼泪。"
> 
> "掌柜的，你能收吗？"

```
（ 陶罐不大，
   却沉甸甸的。
   
   三十年的眼泪......
   得流多少，才能装满这一罐？ ）
```

**阿莲**
> "我想换一样东西。"
> 
> "能让......能让那个老婆子，也尝尝我受的苦。"

```
『 她的眼睛里有恨。
  
  但更多的，是疲惫。 』
```

**【进入关卡提示】**

```
『 要完成这笔交易，
  你需要找到一件能承载怨恨的器皿。
  
  在货架上翻找吧。 』
```

**[进入关卡1]**

---

## 第二幕：探寻

### 关卡1：「破碎的童年」

**目标**：合成【信物链·拨浪鼓】
**难度**：简单（教学关卡性质）
**特殊机制**：无

---

### 剧情碎片A：她叫阿莲

*关卡结束，画面渐暗。*
*出现水墨风格的回忆画面。*

```
『 光绪十一年，江南某镇。
  
  一个瘦小的女孩被牵进一户人家。
  她才十岁，扎着两个羊角辫。
  手里紧紧攥着一个拨浪鼓——
  那是爹娘留给她的唯一念想。 』
```

**画面**：*模糊的人影，小女孩被推进门。*

```
『 "这是你以后的家了。"
  
  爹娘的声音从身后传来，
  越来越远。
  
  她没有回头。
  
  她知道，回头也看不到什么了。 』
```

**【回到当铺】**

**阿莲**
> "我十岁那年被卖到蒋家。"
> 
> "他们说是'童养媳'，'以后就是少爷的媳妇'。"
> 
> *她冷笑了一声。*
> 
> "媳妇？我是他们家的牛马还差不多。"

```
（ 她的手在微微发抖。
   是冷，还是气？ ）
```

**[进入关卡2]**

---

### 关卡2：「日复一日」

**目标**：合成【信物链·染血家书】
**难度**：中等
**特殊机制**：棋盘上有2个「冻疮」障碍，需要合成「药理链·忘忧草」消除

---

### 剧情碎片B：三十年

*关卡结束，画面渐暗。*
*一封染血的家书浮现。*

```
『 这封信她写了三十年。
  
  三更起床，五更睡下。
  洗衣、做饭、喂猪、砍柴。
  
  做得好，没有夸奖。
  做不好，就是一顿打。 』
```

**画面**：*粗糙的手在搓洗衣服，水变成了红色。*

```
『 冬天的井水刺骨，
  她的手裂了又裂，
  结了一层又一层的痂。
  
  有一年除夕，
  她偷偷给远方的爹娘写了封信。
  
  还没写完，就被婆婆发现了。 』
```

**画面**：*信纸被撕碎，散落一地。*

```
『 "写什么写！你爹娘早死了！
   就算没死，也不会来接你！"
   
  那一巴掌，打得她满嘴是血。
  
  血滴在信纸上。
  
  她再也没写过信了。 』
```

**【回到当铺】**

**阿莲**
> "三十年。"
> 
> "我等了三十年。"
> 
> *她的眼眶红了。*
> 
> "老爷死了，少爷也死了。"
> "就剩下那个老婆子。"
> 
> "我熬啊熬，终于熬到她咽气。"
> 
> "可是......"

**阿莲**
> "她死的时候笑了。"
> 
> "她说，'阿莲啊，我这辈子活够本了。'"
> "她说，'你伺候了我一辈子，值了。'"
> 
> *阿莲的声音开始颤抖。*
> 
> "她凭什么......凭什么这么安心！"

```
（ 她的恨意几乎要溢出来了。
   
   三十年的隐忍，
   在这一刻全部爆发。 ）
```

**[进入关卡3]**

---

## 第三幕：真相

### 关卡3：「镇魂之器」

**目标**：合成【巫蛊链·镇魂瓶】
**难度**：较高
**特殊机制**：棋盘上有「封印符」障碍，需要合成「工具链·桃木剑」破除

---

### 剧情碎片C：另一个童养媳

*关卡结束。*
*画面出现了意想不到的场景——*

```
『 镇魂瓶成型的瞬间，
  你看到了一段不属于阿莲的记忆。 』
```

**画面**：*泛黄的旧时光，一个同样瘦小的女孩被推进门。*

```
『 那是五十年前。
  
  同样的小镇，同样的蒋家。
  一个小女孩踉跄着进门。
  
  她也是童养媳。
  
  她的名字，叫阿秀。 』
```

**画面**：*女孩长大，嫁给了蒋家少爷。成为了蒋家的媳妇。*

```
『 阿秀熬成了媳妇，又熬成了婆婆。
  
  有一天，她的丈夫带回来一个小女孩。
  
  "这是给咱儿子的童养媳。"
  
  阿秀看着那个孩子，
  想起了当年的自己。 』
```

*画面定格：老年阿秀的脸，和年轻阿莲的脸，重叠在一起。*

```
『 她本可以对她好一点。
  
  可是她没有。
  
  "我当年受的苦，
   凭什么她不用受？" 』
```

**【回到当铺】**

*阿莲愣住了。*

**阿莲**
> "你说什么......？"
> 
> "她......她也是童养媳？"

```
（ 镇魂瓶静静地躺在柜台上，
   里面隐约浮现两个人影。
   
   一个是阿莲。
   一个是她的婆婆。
   
   两个人的命运，
   竟如此相似。 ）
```

**阿莲**
> "不、不对......"
> 
> "她打我、骂我、折磨我......"
> 
> "她怎么可能......怎么可能是和我一样的人！"

```
『 她在颤抖。
  
  愤怒、迷茫、
  还有一丝......动摇。 』
```

---

## 第四幕：抉择

**【掌柜内心独白】**

```
（ 冤冤相报，何时是头。
   
   阿秀把当年受的苦，
   加倍还给了阿莲。
   
   如果阿莲报复回去，
   这怨恨......还会传给下一个人吗？
   
   可是，
   三十年的苦难，
   难道就这样白白承受了？
   
   掌柜啊掌柜，
   这笔账，你怎么算？ ）
```

**【抉择呈现】**

```
她在等待你的回答。

镇魂瓶就在你手中。
```

---

### 选项A：完成交易

> **「我可以帮你。」**
> 
> *将镇魂瓶交给她。*

---

### 选项B：拒绝交易

> **「放下吧。」**
> 
> *将镇魂瓶收回货架。*

---

### 选项C：另辟蹊径 🔒

> **「让她自己选择。」**
> 
> *需要道具：天机符*

---

## 第五幕：结局

### 结局A：「以怨报怨」

*阿莲接过镇魂瓶。*

**阿莲**
> "谢谢掌柜的。"
> 
> *她的眼神变得冰冷。*
> 
> "我会让她尝尝，什么叫生不如死。"

```
『 她把陶罐放在柜台上，
  转身走进了夜色。
  
  那一罐眼泪，
  被留在了当铺里。
  
  三十年的苦难，
  她选择了遗忘。
  
  从此以后，
  她不会再哭了。
  
  但她也不会再笑了。 』
```

**【收集物品】**
- 「一罐眼泪」入库

**【收集室文案】**
> 这罐眼泪来自一个名叫阿莲的女子。
> 
> 她用三十年的泪水，换来了复仇的机会。
> 
> 据说，她的婆婆在阴间受尽折磨，日夜哀嚎。
> 
> 而阿莲，最终也变成了一个新的"恶婆婆"......

**获得修缮点：⭐⭐⭐**

---

### 结局B：「放下执念」

*阿莲愣住了。*

**阿莲**
> "放下？"
> 
> *她的声音有些颤抖。*
> 
> "三十年的苦，你让我放下？"

**掌柜**（旁白呈现）
```
（ "那三十年，确实苦。"
   "可如果你报复了，又能怎样？"
   "她会痛苦，你会解脱吗？"
   "还是说，你会变成下一个她？" ）
```

*阿莲沉默了很久。*

**阿莲**
> "我......"
> 
> *她突然蹲了下来，抱着头。*
> 
> "我好累啊......"
> 
> "我真的好累啊......"

```
『 她哭了。
  
  不是愤怒的哭，
  是释放的哭。
  
  三十年的委屈，
  在这一刻全部涌了出来。 』
```

*许久之后，她站起身。*

**阿莲**
> "罢了。"
> 
> "就当......就当我这辈子白活了吧。"
> 
> *她擦了擦眼泪，走向门口。*
> 
> "掌柜的，那罐眼泪你留着吧。"
> "就当是......谢礼。"

```
『 她走进了夜色。
  
  第二天清晨，
  当铺门口多了一枝梅花。
  
  寒冬腊月，
  梅花开得正盛。
  
  苦尽......甘来？ 』
```

**【收集物品】**
- 「一罐眼泪」入库
- 「一枝寒梅」入库（隐藏收集）

**【收集室文案】**
> 这罐眼泪来自一个名叫阿莲的女子。
> 
> 她用三十年的泪水，换来了一个选择。
> 
> 她选择了放下。
> 
> 没有人知道她后来去了哪里。
> 只知道第二年春天，镇上新开了一家小茶馆，
> 老板娘的手上有很多旧伤疤，
> 但她总是笑着。

**获得修缮点：⭐⭐⭐（+⭐隐藏奖励）**

---

### 结局C：「轮回之镜」

*需要道具：天机符*

**掌柜**（旁白呈现）
```
（ 也许......
   让她自己看清一切，
   比我替她做决定更好。 ）
```

*你取出天机符，轻轻贴在镇魂瓶上。*

```
『 镇魂瓶亮了。
  
  里面的两个人影越来越清晰。
  
  一个是年轻的阿秀，
  一个是年老的阿秀。
  
  一个在受苦，
  一个在施苦。
  
  同一个人。
  同一个轮回。 』
```

*阿莲看着瓶中的画面，身体僵住了。*

**阿莲**
> "这是......"
> 
> "她年轻时候......也是这样被打的？"
> 
> "她......她哭得和我一样......"

```
『 镇魂瓶里的画面还在继续。
  
  年轻的阿秀熬成了老年的阿秀。
  她看着新来的童养媳——阿莲。
  
  她的眼中闪过一丝犹豫。
  然后......变成了冷漠。
  
  "我当年受的苦，凭什么她不用受？" 』
```

*阿莲的眼泪流了下来。*

**阿莲**
> "原来......原来是这样......"
> 
> "她也恨，她也怨......"
> 
> "但她把这份恨，传给了我......"
> 
> *她看向自己的手。*
> 
> "那我呢？我如果报复了......"
> "下一个阿莲，会恨我吗？"

*她抬起头，看向你。*

**阿莲**
> "掌柜的，我明白了。"
> 
> "这个瓶子......我不要了。"
> 
> *她把陶罐也推了回来。*
> 
> "眼泪也不要了。"
> 
> "我要留着它，提醒自己......"
> 
> "这份苦，到我这里为止。"

```
『 她深深地鞠了一躬，
  转身离去。
  
  门外，天已经亮了。
  
  三十年的阴霾，
  终于散去。 』
```

**【收集物品】**
- 「一罐眼泪」入库
- 「破碎的镇魂瓶」入库（她亲手打碎的）
- 「阿莲的拨浪鼓」入库（隐藏收集——她找回了童年的自己）

**【收集室文案】**
> 这罐眼泪来自一个名叫阿莲的女子。
> 
> 她本可以用它换取复仇，但她选择了打破轮回。
> 
> 三年后，有人在镇上看到她。
> 她开了一家小学堂，专收没人要的女娃娃。
> 
> 她说："我希望她们的眼泪，都是幸福的眼泪。"

**获得修缮点：⭐⭐⭐⭐⭐（完美结局）**
