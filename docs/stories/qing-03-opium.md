# 清末·故事3：烟鬼

> 风格标签：`#微恐` `#反转` `#社会批判`
> 时长预估：20-25分钟
> 关卡数量：4关

---

## 核心设定

**表面故事**：儿子阿诚想救染上烟瘾的父亲。

**真相**：父亲早就死了。阿诚救的，是一个"别的东西"。

**社会隐喻**：鸦片不只是毒品，它是一种"寄生"——寄生在人身上，寄生在家庭上，寄生在整个国家上。那些烟馆里的"人"，还是人吗？

---

## 第一幕：入店

### 场景：深夜的当铺

**【环境渲染】**

```
『 亥时三刻。

  街上飘着一股甜腻的气息——
  从对面的烟馆传来。
  
  那气味像是腐烂的花，
  闻久了会上瘾。
  
  风铃响了。
  
  但门口......没有人。 』
```

**【客人登场】**

*你往门外看——*

*一个年轻人跪在门槛外，额头贴着地。*
*像是跪了很久。*

```
『 他不敢进来。
  
  或者说——
  他在等你允许。 』
```

**掌柜**
> "进来吧。"

*年轻人抬起头，眼眶通红。*
*他爬着进来，不敢站。*

**【初次对话】**

**阿诚**
> "掌柜的......救命......"
> 
> *声音沙哑。*
> 
> "求您救救我爹......"

```
（ 又是大烟。
   这条街上，十户有九户被害过。 ）
```

**阿诚**
> "我爹......抽了十年了......"
> 
> "我想带他回家，可他......他不认我了......"
> 
> *突然抬起头，眼神有些疯狂。*
> 
> "他看着我，叫我'小鬼'。"
> 
> "他说——'你不是我儿子，我儿子早死了'。"

```
（ ......

   脑子烧坏了也正常。
   
   但"早死了"是什么意思？ ）
```

**阿诚**
> "掌柜的，我不要他戒烟了。"
> 
> "我只想......我只想让他认得我。"
> 
> "哪怕一次。"
> 
> *他从怀里掏出一样东西。*
> 
> "这是我娘的嫁妆，全在这儿了。您收了吧。"
>

## 第二幕：烟馆

### 关卡1：「迷烟」

**目标**：合成【工具链·照妖镜】
**难度**：简单
**特殊机制**：棋盘笼罩在"烟雾"中，部分格子看不清

---

### 剧情碎片A：烟馆夜景

*关卡结束，画面切换——*
*你跟着阿诚来到对街的烟馆。*

```
『 "福寿阁"。
  
  三个字用金漆写的，
  在夜色里泛着诡异的光。
  
  门口挂着灯笼，
  红得像血。 』
```

**画面**：*推开门，一股浓重的甜香扑面而来。*

```
『 烟雾缭绕。
  
  你几乎看不清里面的人。
  
  只能看到一个个......
  
  影子。 』
```

*你仔细看——*

*那些"影子"躺在床榻上，手里握着烟枪。*
*有的在吞云吐雾。*
*有的一动不动。*
*有的......在笑。*

*但那笑容不对劲。*

```
『 那笑容......
  
  不是人笑的样子。
  
  嘴角咧得太开，
  眼睛却是空的。
  
  像是......
  
  被什么东西占据了身体，
  正在享用着。 』
```

**阿诚**（在你身边，压低声音）
> "那边......第三张床......"
> 
> "那是我爹。"

*你顺着他指的方向看去——*

*一个瘦骨嶙峋的老人躺在那里。*
*皮包骨，眼窝深陷。*
*手指像枯枝，死死攥着烟枪。*

```
『 他的眼睛是睁着的。
  
  但里面......什么都没有。
  
  像是两个空洞。 』
```

**阿诚**
> "爹......"
> 
> *他走过去，跪在床边。*
> 
> "爹，是我，阿诚啊......"

*老人慢慢转过头。*

*盯着阿诚看了很久。*

**林掌柜**
> "......你是谁？"
> 
> *声音像破风箱。*
> 
> "又来骗我钱的？"
> 
> "滚。"

**阿诚**
> "爹！我是阿诚！您儿子啊！"

*老人突然笑了。*

*那笑声让你起了一身鸡皮疙瘩。*

**林掌柜**
> "儿子？"
> 
> "我儿子早死了。"
> 
> "十年前就死了。"
> 
> *他重新躺下，看着天花板。*
> 
> "你是小鬼。来找我索命的小鬼。"
> 
> "我知道的......"

```
『 阿诚愣在那里。
  
  "十年前就死了"......
  
  十年前发生了什么？ 』
```

**[进入关卡2]**

### 剧情碎片B：十年前

*关卡结束。*
*照妖镜亮了——你看到了十年前的画面。*

```
『 光绪十二年。
  
  济世堂药铺后院。
  
  一个五岁的男孩在玩耍。 』
```

**画面**：*小阿诚在院子里追蝴蝶。*

**林夫人**（远处喊）
> "阿诚！别跑远了！"

**小阿诚**
> "娘！我抓到蝴蝶了！"

*画面一转——*

*药铺前厅。*
*林掌柜正在和一个洋人说话。*

**洋人**（通译转述）
> "林掌柜，这是最新的货，比上次的纯。"
> 
> "先尝尝，满意的话，下次多拿点。"

*林掌柜接过一个小瓷瓶，打开，闻了闻。*

**林掌柜**
> "这次......这次要多少银子？"

**洋人**
> "老规矩。"
> 
> *笑得意味深长。*
> 
> "家里有什么值钱的，都可以。"

*林掌柜想了想，转身进了后院。*

```
『 他没去拿银子。
  
  他去了......
  
  儿子玩耍的地方。 』
```

**画面**：*林掌柜弯下腰，对小阿诚笑。*

**林掌柜**
> "阿诚，跟爹走，爹带你去个好玩的地方。"

**小阿诚**
> "什么地方？"

**林掌柜**
> "去了就知道了。"

*他牵着儿子的手，走向前厅。*

*小阿诚蹦蹦跳跳地跟着。*

*不知道自己正在被带向哪里。*

```
『 画面在这里断了。
  
  但你已经猜到了。
  
  十年前，
  林掌柜为了换大烟——
  
  把儿子卖了。 』
```

**【回到当铺——反转】**

*你从回忆中醒来。*

*看向跪在地上的阿诚。*

**掌柜**
```
（ 等等。
   
   如果阿诚十年前被卖了......
   
   眼前这个人是谁？ ）
```

*你仔细看阿诚的脸——*

*年轻、清秀、五官端正。*

*但......*

```
『 他的影子不对。
  
  灯光下，
  他的影子比他本人......
  
  小得多。
  
  像是一个孩子的影子。 』
```

**阿诚**（发现你在看他，笑了）
> "掌柜的，怎么了？"
> 
> *那笑容......*
> 
> *和烟馆里那些"人"的笑容一模一样。*

**[进入关卡3]**

## 第三幕：真相

### 关卡3：「还魂」

**目标**：合成【巫蛊链·镇魂瓶】
**难度**：较高
**特殊机制**：
- 棋盘上有「小孩影子」在游荡，会"偷走"你的棋子
- 如果被偷太多，关卡失败

---

### 剧情碎片C：阿诚的真实身份

*关卡结束。*
*镇魂瓶成型，映出了"阿诚"的真身——*

```
『 镜中不是一个年轻人。
  
  而是一个......
  
  残缺的孩子。
  
  左眼是空的，一个黑洞。
  右手齐腕断了，伤口狰狞。
  双腿扭曲成不可能的角度。
  嘴巴张开——
  
  没有舌头。 』
```

*"阿诚"看着镜中的自己，笑了。*

*那笑容......没有声音。*

*因为他发不出声。*

**"阿诚"**（声音直接在你脑子里响起）
> "掌柜的，你看到了呀。"
> 
> *稚嫩的童声，却带着阴冷。*
> 
> "丑吗？"
> 
> "这是他们弄的。"

```
『 他是真的阿诚。
  
  五岁那年被卖掉的阿诚。
  
  被卖去了哪里？
  
  卖给了......
  采生折割的乞丐头子。 』
```

**"阿诚"**
> "他们说，残了的孩子好要饭。"
> 
> "眼睛挖了，路人会可怜。"
> 
> "手脚断了，跑不掉。"
> 
> "舌头割了，喊不出来。"
> 
> *他歪着头，空洞的眼眶对着你。*
> 
> "我在街上爬了三年。"
> 
> "冬天冷，夏天热，饿了吃剩饭，渴了喝脏水。"
> 
> "最后死在城隍庙门口。"
> 
> "没人收尸，野狗来啃。"
> 
> *笑了。*
> 
> "我爹呢？我爹在烟馆里抽得快活。"

```
『 林掌柜为了换大烟，
  把五岁的儿子卖给了采生折割的乞丐头子。
  
  孩子被弄残，
  被迫乞讨，
  八岁那年冬天冻死在街头。
  
  死的时候，
  手里还攥着一个破碗——
  那天他只讨到三文钱。
  
  他的怨念没有散。
  
  十年后，
  他找到了父亲。 』
```

**"阿诚"**
> "掌柜的，我不是来求你救他的。"
> 
> *笑容不变。*
> 
> "我是来求你......帮我把他带走。"
> 
> "让他陪我。"
> 
> "在下面，永远陪我。"

```
『 一个被父亲卖掉的孩子。
  一个为了毒品杀死儿子的父亲。
  
  这笔账......
  怎么算？ 』
```

**[进入关卡4]**

### 剧情碎片D：烟馆的秘密

*关卡进行中，你看到了更多画面——*

```
『 那个洋人不只是卖大烟。
  
  他还......收人。
  
  烟鬼们付不起钱，
  就用别的东西换——
  
  妻子。
  女儿。
  儿子。
  
  福寿阁后面有一个地窖。
  
  那里面......
  
  堆满了小衣服。 』
```

*画面闪过：一件件童装，有的染着血。*

```
『 那些被卖掉的孩子，
  有的成了矿工。
  有的成了暗娼。
  有的......被采生折割。
  
  而那些烟鬼父母，
  抽完最后一口烟，
  什么都不记得了。
  
  大烟会让人忘记一切。
  
  包括——
  自己曾经是个人。 』
```

**画面**：*福寿阁的老板，一个肥胖的中年人，正在和洋人分账。*

**烟馆老板**
> "今天又收了三个。"
> 
> "两个女娃，一个男娃。"
> 
> "林家那个最好，养得白白胖胖。"
> 
> *笑。*
> 
> "城西丐头出大价钱。说是好苗子，弄残了能用好几年。"

**洋人**
> "Good. Business is good."
> 
> "继续。"
> 
> "这些黄猴子，只要有烟，什么都肯卖。"

```
『 这不只是一个家庭的悲剧。
  
  这是一条产业链——
  
  洋人卖烟，
  烟馆收人，
  丐头买命。
  
  而整个大清，
  从官府到百姓，
  都在装聋作哑。
  
  因为——
  
  谁不抽两口呢？ 』
```

**【回到当铺】**

*小阿诚——那个鬼——站在你面前。*

*他的身后，站着更多的影子。*

*都是孩子。*

*都在笑。*

**"阿诚"**
> "掌柜的，他们都是我的朋友。"
> 
> "我们都是被爹娘卖掉的。"
> 
> *他伸出手。*
> 
> "帮帮我们吧。"
> 
> "让那些卖掉我们的人......来陪我们。"

```
『 那些影子越来越多。
  
  不只是孩子。
  还有女人。
  还有老人。
  
  都是被大烟害死的人。
  
  他们在等一个答案。 』
```

## 第四幕：抉择

**【掌柜内心独白】**

```
（ 林掌柜该死吗？
   当然该死。
   
   他为了大烟，卖掉了亲生儿子。
   禽兽不如。
   
   可是......
   
   只杀他一个，有用吗？
   
   烟馆还在。
   洋人还在。
   那些买孩子的人还在。
   
   杀一个烟鬼，
   改变不了任何事。
   
   但不杀，
   这些冤魂怎么办？ ）
```

**【抉择呈现】**

```
小阿诚站在那里，
身后是无数亡魂。

他们在等你的回答。
```

---

### 选项A：「偿命」

> **「冤有头债有主。林掌柜欠你一条命，就让他还。」**
> 
> *以命抵命*

---

### 选项B：「超度」

> **「你已经死了。执念只会让你更痛苦。放下吧。」**
> 
> *引导他们往生*

---

### 选项C：「连根拔起」 🔒

> **「杀一个烟鬼有什么用？要杀，就把这条根都拔了。」**
> 
> *需要道具：天机符*

## 第五幕：结局

### 结局A：「父债子偿」

*你将镇魂瓶交给小阿诚。*

**掌柜**
> "去吧。他欠你的，让他还。"

*小阿诚接过瓶子，笑了。*

**"阿诚"**
> "谢谢掌柜的。"
> 
> *他转身，走向烟馆。*
> 
> "爹，我来接你了。"

```
『 那天夜里，
  福寿阁传出一声惨叫。
  
  第二天，
  人们发现林掌柜死在了床榻上。
  
  面容扭曲，双手掐着自己的脖子。
  
  像是被什么东西......勒死的。
  
  奇怪的是，
  他的脸上带着笑。
  
  空洞的笑。 』
```

*烟馆里的其他烟鬼们看着这一幕，没有任何反应。*
*他们继续抽烟。*
*像是什么都没发生过。*

```
『 林掌柜死了。
  
  但福寿阁还开着。
  洋人还在卖烟。
  孩子还在被卖。
  
  什么都没有改变。
  
  只是......
  
  多了一个鬼而已。 』
```

**【收集物品】**
- 「烟枪」入库（林掌柜死时还攥着）
- 「一件染血小衣」入库

**【收集室文案】**
> 这件小衣属于一个五岁的男孩。
> 
> 他的父亲为了一口烟，把他卖给了炼丹的方士。
>
> 十年后，他回来找父亲。
> 
> 父亲终于认出了他——在死的那一刻。

**获得修缮点：⭐⭐**

### 结局B：「往生」

*你点燃长明灯。*

**掌柜**
> "孩子，你死了，他也快死了。"
> 
> "再执着下去，你们都不得解脱。"

*小阿诚愣住了。*

**"阿诚"**
> "可是......可是他卖了我......"
> 
> *声音变得委屈。*
> 
> "我好痛......我死的时候好痛......"

**掌柜**
> "我知道。"
> 
> *声音平静。*
> 
> "但你杀了他，就不痛了吗？"
> 
> "你会变成和他一样的东西——被恨吞噬的怪物。"

*小阿诚低下头。*

*身后那些影子也安静了。*

**"阿诚"**
> "我不想当怪物......"
> 
> *小小的声音。*
> 
> "我只是......想让他叫我一声阿诚......"
> 
> "他从来没有好好叫过我的名字......"

```
『 你将长明灯递给他。
  
  灯光照亮了他的脸——
  
  五岁的小脸，
  满是泪痕。 』
```

**掌柜**
> "走吧。去投胎。"
> 
> "下辈子，找一个会好好叫你名字的爹娘。"

*小阿诚接过灯，看了很久。*

**"阿诚"**
> "好。"
> 
> *他转身，带着那些影子，走向光里。*
> 
> "下辈子......我想当有爹娘疼的孩子......"

```
『 他们消失了。
  
  当铺重新安静下来。
  
  烟馆还在。
  林掌柜还在抽烟。
  
  什么都没有改变。
  
  但至少......
  
  那些孩子不用再痛了。 』
```

*一个月后。*

```
『 林掌柜死了。
  
  死在烟榻上，
  手里还攥着烟枪。
  
  死的时候，
  没有人在身边。
  
  入殓的人说，
  他的脸上......
  好像有泪痕。
  
  没人知道他在哭什么。
  也没人在乎。 』
```

**【收集物品】**
- 「长明灯（空壳）」入库
- 「一滴鬼泪」入库（小阿诚留下的）

**【收集室文案】**

## 设计笔记

### 反转层次

1. **第一层**：儿子想救父亲 → 父亲不认儿子
2. **第二层**：父亲说"儿子死了" → 原来真的死了
3. **第三层**：阿诚是鬼 → 他是被父亲卖掉害死的
4. **第四层**：不只是一个家庭 → 是整条产业链

### 微恐元素

- 烟馆里"人"的笑容不对劲
- 阿诚的影子是小孩的
- 地窖里的童装和骨头
- "被什么东西占据了身体"的暗示

### 社会批判

这个故事的核心不是"一个坏父亲"，而是：

1. **殖民者的毒害**：洋人不只是卖毒品，还在收割人命
2. **产业链的恶**：烟馆→人贩子→采生折割/矿主/妓院，形成完整闭环
3. **沉默的帮凶**：官府睁一只眼闭一只眼，因为"谁不抽两口"
4. **反抗的代价**：烧烟馆的人被当成"刁民"杀了
5. **历史的正名**：但一百年后，我们知道谁是对的

### 与前两个故事的呼应

| 故事 | 受害者 | 加害者 | 真正的敌人 |
|------|--------|--------|------------|
| 童养媳 | 阿莲 | 婆婆 | 制度（童养媳陋习） |
| 纸扎匠 | 父子 | 时代 | 洋货冲击传统 |
| 烟鬼 | 孩子们 | 父亲 | **殖民侵略+系统性腐烂** |

三个故事逐层递进：
- 个人悲剧 → 家庭悲剧 → 社会悲剧
- 受害者视角 → 亲情纠葛 → 群体控诉
