# 当铺修缮系统

参考 Royal Match 城堡修缮模式，每个时代当铺有**多个修缮区域**，完成故事订单获得**修缮点**，逐步修缮当铺。

---

## 核心流程

```
完成订单关卡 → 获得修缮点(⭐) → 选择修缮区域 → 观看修缮动画 → 解锁新故事/新区域
```

---

## 🏮 清末当铺

**5个故事，12个修缮点**

| 区域 | 修缮前 | 修缮后 | 所需⭐ |
|------|--------|--------|--------|
| **大门** | 破败木门，门神剥落 | 朱漆大门，新门神 | 2 |
| **柜台** | 布满刀痕，算盘散落 | 红木柜台，铜算盘 | 3 |
| **货架** | 蛛网缠绕，落满灰尘 | 整齐陈列，分类清晰 | 2 |
| **神龛** | 歪斜破损，香炉生锈 | 端正庄严，香火不断 | 2 |
| **地板** | 血迹斑驳，木板腐朽 | 青砖铺地，干净整洁 | 3 |

### 解锁顺序
1. 完成第1个故事 → 解锁大门修缮
2. 完成第2个故事 → 解锁柜台+货架修缮
3. 完成第3个故事 → 解锁神龛修缮
4. 完成第4个故事 → 解锁地板修缮
5. 完成第5个故事 → **时代转换动画** → 进入民国

## 修缮的微恐细节

每个修缮完成后，新物件都带有**微妙的恐怖元素**：

| 物件 | 恐怖细节 |
|------|----------|
| 留声机 | 偶尔自己播放一首没有的唱片 |
| 月份牌 | 美女的眼睛会跟着你转 |
| 老电话 | 半夜响起，接起来是沙沙声 |
| 吊扇 | 有时会逆向转动 |
| 监控 | 回放里，柜台后面多了一个人影 |
| WiFi | 名字有时变成"他在看着你" |

## 收集室：执念陈列馆

当铺内设有一个**特殊房间**，存放被典当的"特殊物品"：

| 物品 | 来源故事 | 点击文案 |
|------|----------|----------|
| 一罐眼泪 | 清末·童养媳 | 「三十年的泪水，终于流尽了。」 |
| 一段笑声 | 民国·花旦 | 「台上笑，台下哭，戏里戏外分不清。」 |
| 半张全家福 | 现代·空巢老人 | 「另一半在儿子那里，但他再也不会看了。」 |
| 一枚咬过的戒指 | 民国·负心汉 | 「恨到咬碎银牙，也换不回那个人。」 |

每件物品都可以点击查看背后的微恐小故事，增加收集动力。
