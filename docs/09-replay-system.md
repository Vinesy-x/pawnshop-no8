# 结局回放系统

---

## 设计目标

1. **不破坏首次体验的沉浸感**
2. **让玩家有动力探索所有结局**
3. **平滑自然，不显得突兀**
4. **合理的商业化切入点**

---

## 方案：「因果簿」系统

### 概念包装

当铺里有一本**「因果簿」**，记录着每一笔交易的来龙去脉。

> *"每一个选择，都会分出不同的因果。*
> *因果簿记录了所有可能发生的事。*
> *但你只能亲眼见证其中一条路。*
> *除非......你愿意付出一点代价。"*

---

## 玩家体验流程

### 首次通关
```
完成故事 → 根据选择看到结局A → 故事结束
                ↓
        结局B/C 显示为「迷雾」状态
                ↓
        因果簿记录：「此人的命运，还有其他可能......」
```

### 回顾入口

故事完成后，在以下位置可以进入回放：

1. **收集室** - 点击该故事的收集物品
2. **因果簿** - 主界面的独立入口
3. **故事完成时的弹窗提示**

---

## 回放机制

### 方式1：「窥探因果」（免费但有限制）

**从抉择点继续**，不用重打关卡。

```
┌─────────────────────────────────────────┐
│                                         │
│   📖 因果簿 · 童养媳的眼泪              │
│                                         │
│   你已见证的因果：                       │
│   ✓ 结局A：以怨报怨                     │
│                                         │
│   未见证的因果：                         │
│   🔒 结局B：放下执念                     │
│   🔒 结局C：轮回之镜                     │
│                                         │
│   ─────────────────────────             │
│                                         │
│   [👁️ 窥探因果]  消耗：1个因果之眼       │
│                                         │
│   或                                     │
│                                         │
│   [📺 观看广告]  免费获得1次窥探机会      │
│                                         │
└─────────────────────────────────────────┘
```

**「因果之眼」获取途径**：
- 每日登录赠送 1 个
- 完成成就获得
- 付费购买（6元=5个）
- 看广告（每日限3次）

---

### 方式2：「命运重演」（完整重玩）

想要**重新体验完整故事**（包括关卡），有两种方式：

#### A. 免费重玩（消耗体力）
- 正常消耗体力重打关卡
- 适合想重温剧情的玩家

#### B. 「时光倒流」（跳过关卡）
```
┌─────────────────────────────────────────┐
│                                         │
│   🕰️ 时光倒流                           │
│                                         │
│   回到故事开始，                         │
│   但你可以跳过已通过的关卡。             │
│                                         │
│   消耗：1个轮回沙漏                      │
│                                         │
│   [确认]  [取消]                         │
│                                         │
└─────────────────────────────────────────┘
```

**「轮回沙漏」获取途径**：
- 完成一个时代所有故事后赠送 3 个
- 付费购买（12元=3个）
- 稀有成就奖励

## 隐藏结局的特殊处理

### 结局C（需要道具）的解锁提示

如果玩家第一次没有道具「天机符」，会显示：

```
┌─────────────────────────────────────────┐
│                                         │
│   🔒 另辟蹊径                            │
│                                         │
│   "似乎还有别的办法......               │
│    但你需要一件特殊的东西。"             │
│                                         │
│   需要：天机符 ×1                        │
│   当前：0                                │
│                                         │
│   [获取天机符]                           │
│                                         │
└─────────────────────────────────────────┘
```

**「天机符」获取途径**：
- 完成特定成就（如：解锁3个故事的B结局）
- 修缮当铺到特定阶段
- 付费购买（18元=1个）
- **不能通过广告获取**（保持稀有性）

## 成就激励

### 「因果收藏家」系列成就

| 成就 | 条件 | 奖励 |
|------|------|------|
| 初窥因果 | 解锁任意故事的第2个结局 | 因果之眼 ×3 |
| 命运交织 | 解锁任意故事的全部结局 | 轮回沙漏 ×1 |
| 清末百态 | 解锁清末全部故事的全部结局 | 天机符 ×1 |
| 因果大师 | 解锁全部故事的全部结局 | 专属称号 + 隐藏内容 |

---

## 数值平衡

### 免费玩家体验
- 每个故事首次必定能完整体验1个结局
- 每日可免费窥探1-2个其他结局（登录+广告）
- 约1周可解锁一个故事的全部结局

### 付费玩家体验
- 可以立即解锁所有想看的结局
- 「因果通行证」月卡：28元/月，每日免费窥探3次

## UI/UX 细节

### 迷雾结局的展示
- 结局名称模糊处理，只能看到轮廓
- 悬停时显示「？？？」
- 有微微的光芒吸引点击

### 解锁动画
- 迷雾散去的动画
- 配合神秘的音效
- 结局名称逐字显现

### 已解锁结局的标记
- 显示完整名称
- 可以随时重看（免费）
- 标记「已收藏」
