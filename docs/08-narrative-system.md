# 故事叙事系统

---

## 故事模版结构

每个故事分为 **5个阶段**，通过「对话-关卡-对话」的节奏推进：

```
┌─────────────────────────────────────────────────────────┐
│  第一幕：入店                                            │
│  ├─ 环境渲染（当铺氛围）                                 │
│  ├─ 客人登场（形象描写）                                 │
│  └─ 初次对话（提出请求，但信息不全）                     │
├─────────────────────────────────────────────────────────┤
│  第二幕：探寻                                            │
│  ├─ 【关卡1】简单难度                                    │
│  ├─ 剧情碎片A（揭示部分背景）                            │
│  ├─ 【关卡2】中等难度                                    │
│  └─ 剧情碎片B（揭示更多细节）                            │
├─────────────────────────────────────────────────────────┤
│  第三幕：真相                                            │
│  ├─ 【关卡3】较高难度                                    │
│  ├─ 剧情碎片C（关键转折/微恐揭示）                       │
│  └─ 完整真相浮现                                         │
├─────────────────────────────────────────────────────────┤
│  第四幕：抉择                                            │
│  ├─ 掌柜内心独白（引导玩家思考）                         │
│  ├─ 呈现选项（2-3个选择）                                │
│  └─ 玩家做出决定                                         │
├─────────────────────────────────────────────────────────┤
│  第五幕：结局                                            │
│  ├─ 根据选择展现不同结局                                 │
│  ├─ 客人离去                                             │
│  ├─ 获得修缮点                                           │
│  └─ 收集物品入库（可回顾）                               │
└─────────────────────────────────────────────────────────┘
```

## 剧情碎片呈现方式

### 方式1：闪回画面
关卡结束后，屏幕渐暗，出现一段「回忆画面」：
- 水墨风格/老照片风格
- 配合旁白文字
- 可以有简单动画（人影走动、物品掉落）

### 方式2：物品独白
合成出关键物品时，物品会「说话」：
> *染血家书在你手中微微发烫。*
> *你听到了遥远的哭声......*

### 方式3：环境变化
当铺背景随剧情变化：
- 灯光明暗
- 出现幻象
- 物品位置改变
