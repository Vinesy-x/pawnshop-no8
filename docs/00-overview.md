# 无名当铺 · 概览

> 「当掉执念，换取解脱」
> 
> 中式恐怖 × 二合炼化 × 时代变迁

---

## 游戏简介

你是**无名当铺**的掌柜，这家店坐落在阴阳交界处，跨越百年时光，只在"需要它的人"面前显现。

来客不是为了钱，而是为了**平息生前的遗憾**。他们典当的是执念，换取的是解脱。

---

## 核心循环

```
客人入店 → 讲述故事 → 提出订单
    ↓
二合消除关卡 → 合成所需物品
    ↓
完成交易 → 剧情分支/抉择
    ↓
获得修缮点 → 修缮当铺
    ↓
解锁新故事 → 推进时代
```

## 设计理念

### 恐怖但不恶心
- 心理恐惧 > 视觉刺激
- 氛围营造 > 血腥画面
- 细思极恐 > 跳吓惊吓

### 有泪点不煽情
- 留白让玩家自行感受
- 悲剧不是目的，是触动的手段
- 每个故事都有"救赎"的可能

### 游戏性与叙事平衡
- 消除不是填充，是"整理委托人的执念"
- 难度曲线跟随剧情情绪
- 关键剧情点设计专属关卡机制
